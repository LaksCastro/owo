## Clone project

### 1. Make sure your Workspace is fine to Flutter Development, if not, first [check this here](https://flutter.dev/docs/get-started/install)

```
flutter doctor
```

### 2. Clone the repository

```
git clone https://github.com/LaksCastro/zero.git && cd zero
```

### 3. Configure Firebase for Android  

Put your Firebase file config named google-services.json in:
```
~/android
  /app
   /src
   build.gradle
   # now, put here the firebase-services.json, you can get this file in Firebase Console
```

### 4. Configure Firebase for IOS  

Put your Firebase file config named google-services.json in:
```
~/ios
  /Runner
   ...
   /Base.lproj
   AppDelegate.swift
   Info.plist
   # now, put here the GoogleService-Info.plist, you can get this file in Firebase Console
```


### 5. Run debug mode

Press F5 if VsCode or run:
```
flutter run
```

### 6. To generate .apk

```
flutter build apk --split-per-abi
```

### 7. To generate .ipa

I still have no way to emulate an IOS on my machine, as I don't have any MACOS or an Apple Smartphone. So I'm in the dark about IOS, but I took triple care when installing packages and keeping everything working for IOS, so this command should be working:
```
flutter build ios
```
